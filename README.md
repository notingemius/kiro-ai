# Kiro AI — Reply Helper

Chrome-расширение (Manifest V3). Выдели текст на любой странице — справа сверху появится кнопка **K**. Клик → панель с подсказкой от AI:

- **Тема ответа** — одной фразой
- **План** — шаги, по которым строится ответ
- **Шаблон ответа** в четырёх частях:
  - **Реакция** на его слова
  - **Эмоция / наблюдение** от себя
  - **Кусочек себя** — личный штрих, опыт, мысль
  - **Вопрос, который копает глубже**

Копируешь шаблон, дописываешь своё, отправляешь. Отвечаешь сам — AI не заменяет тебя, а помогает сформулировать живо и по-человечески.

## Установка (как разработчик)

1. Открой в Chrome: `chrome://extensions`
2. Включи «Режим разработчика» (правый верхний угол)
3. «Загрузить распакованное расширение» → выбери папку `kiro-ai/`
4. Кликни на иконку расширения → откроется страница настроек
5. Вставь API-ключ OpenRouter (https://openrouter.ai/keys) и выбери модель
6. Сохрани

## AI-провайдер

Используется [OpenRouter](https://openrouter.ai/) — там есть **бесплатные** модели (суффикс `:free`). Карта при регистрации не нужна. Лимиты free-моделей: ~20 запросов / мин, ~50 / день.

Рекомендуемые бесплатные модели:

- `openai/gpt-oss-120b:free` — умная, дефолт
- `openai/gpt-oss-20b:free` — быстрее
- `nvidia/nemotron-nano-9b-v2:free` — очень быстрая
- `deepseek/deepseek-chat-v3.1:free`
- `z-ai/glm-4.5-air:free`
- `meta-llama/llama-3.3-70b-instruct:free`
- `qwen/qwen3-235b-a22b:free`
- `google/gemma-3-27b-it:free`

Можно вписать свой slug в настройках.

## Как пользоваться

- Выдели текст мышкой → кликни на появившуюся кнопку **K** (или ПКМ → «Kiro AI: подсказать ответ»)
- Панель покажет подсказку
- «Скопировать шаблон» → вставь в чат, поправь под себя, отправь
- «Ещё вариант» → попросить другой вариант
- `Esc` — закрыть

## Структура

```
kiro-ai/
├── manifest.json          # MV3 манифест
├── icons/                 # 16/48/128 PNG + SVG исходник
├── scripts/gen_icons.py   # генератор иконок (stdlib, без Pillow)
└── src/
    ├── background.js      # service worker: вызов OpenRouter API
    ├── content.js         # обработка выделения + UI на странице
    ├── content.css        # стили триггера и панели
    ├── options.html       # страница настроек
    ├── options.js         # логика настроек
    └── options.css        # стили настроек
```

## Приватность

- Ключ хранится в `chrome.storage.sync` — не уходит никуда, кроме как в openrouter.ai.
- Выделенный текст отправляется только на OpenRouter.
- На бесплатных `:free` моделях провайдеры могут использовать запросы для обучения. Не вставляй туда чувствительную переписку.
